What are Web Forms?

Interactive pages where inputs are taken from the users to register, give inputs to the Application to perform CRUD operations within a Web Application. In most of the cases, Web apps have 80% of content which are read only. Only a few of the documents within a Web site can expect to take inputs from the users. These inputs are in the form of FORMS. Forms are usually processed in a Web server. The processing is done using high level programming languages like PHP, JSP, ASP.NET and Perl.
However after Nodejs came into picture, JavaScript is also used for processing Web Forms.
